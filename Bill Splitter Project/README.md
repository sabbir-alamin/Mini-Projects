# Bill Splitter Calculator

A simple Python script that calculates a restaurant bill, adds a tip, and splits the total equally among a group of friends.

## Features

- Calculates total bill from appetizers, main courses, desserts, and drinks
- Automatically applies a 25% tip
- Splits the final bill evenly among all friends
- Rounds each person's share to 2 decimal places

## How to Run

```bash
python bill_splitter.py
```

## Example Output

```
Total bill so far: 198.83
Tip amount: 49.7075
Total with tip: 248.5375
Bill per person: 62.134375
Each person pays: 62.13
```

## Requirements

- Python 3.x (no external libraries needed)

## Possible Improvements

- Accept user input for number of friends and order items
- Allow custom tip percentage
- Handle uneven splits (e.g., one person orders more)
