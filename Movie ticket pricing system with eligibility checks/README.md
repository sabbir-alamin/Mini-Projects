# Ticket Booking System (Python)

A simple Python script that simulates a movie/event ticket booking system. It checks age eligibility, applies membership discounts, calculates extra charges for weekend/evening shows, and computes the final ticket price based on seat type.

## Features

- Age eligibility check for booking and evening shows
- Membership discount logic
- Weekend / evening show extra charges
- Seat-type based service charges (Premium, Gold, Standard)
- Final price calculation

## How to Run

```bash
python ticket_booking.py
```

## Example Output

```

Discount: 0
Extra charges will be applied

Service charges: 3

```

## Requirements

- Python 3.x (no external libraries needed)
